Research Python

Research the history, use and future of the Python programming language.

1. What was the motivation for creating Python?
-  To create general purpose code related to data analysis and visualization.

2. What are the strengths of the Python language?
- Easy to learn, can be used for almost any light weight issue.

3. What are the weaknesses or issues with the Python language?
- Slower than other compiled languages.

4. How and why is Python being used in today's development environment?
- It is used for data analytics, cloud developing, and web devlopment

5. What does the job market look like for Python skills?
- Full Stack Web Developer, Cloud Developer, Data Engineer, and a lot more.
